# 事業モデル（商品企画スキル共通コンテキスト）

商品企画系スキル（demand-research / product-idea / idea-kill-test / market-research / product-planning）は、このファイルを前提知識として最初に読むこと。

## ビジネスモデル
- D2C EC。SNS広告（Instagram/TikTokのフィード・リール広告）→ LP → 購入の直販モデル
- 商品単価: ¥10,000〜¥20,000 / 原価率: 約30% / 広告費比率: 約40%
- 突然SNS広告で知った商品を、決して安くない価格で衝動購入してもらうモデル
- 年間3,000万円以上の売上を目指す
- 参考企業: amepla.jp（株式会社アメイズプラス。整体パンプス、クッション型トランポリン等）

## このモデルで売れる商品の絶対条件
1. **商品＝広告クリエイティブ**: スクロールを止めるフックと「納得の理屈」が商品自体に内蔵されていること。LPのFVが書けない商品は、企画として存在しないのと同じ
2. **深い悩みの解決**: 日常的・慢性的な悩みにアプローチする（肩こり、睡眠、ダイエット、美容、腰痛、姿勢 等）
3. **独自性**: Amazon/楽天の¥3,000類似品に流れない理由があること
4. **3秒の映像化**: リール/TikTokで効果・使用感が3秒で伝わること

## 価格設定原則
- 販売価格は「売りやすい無難な価格」ではなく、**顧客が買う可能性のある最大値（支払い意思の上限）**を狙う
- 根拠は3点で組み立てる:
  1. **代替手段の累計コスト**（例: マッサージ¥6,000×月1回×1年＝¥72,000。それと比べた割安感）
  2. **競合の最高価格帯**とその売れ行き（高くても売れている実績＝上限の証拠）
  3. **悩みの深さ**（本音の切実さ。切実なほど上限は上がる）
- 広告費40%のモデルでは粗利が生命線。迷ったら価格を下げるのではなく、価格に見合う「納得の理屈」を強くする
- **価格を単体で書かない**。必ず「なぜこの価格でいけると考えるか」の根拠（上記3点の実データ）とセットで提示する

## 収益構造
- ROAS = 売上 ÷ 広告費
- 売上 = アクセス数 × CVR × 客単価 / アクセス数 = 広告費 ÷ CPC
- CPC目安: Instagram/Facebook ¥50〜150、TikTok ¥30〜100
- CVR目安: 1〜3%（D2C EC平均）

## 市場実測シグナル（机上の定数より、ネットで観測できる現実を優先）
販売数・経済性・ターゲット量の推定は、教科書的なベンチマーク定数ではなく、ネットで観測できる実測シグナルを一次根拠にする:
1. **広告の継続配信期間**: Meta広告ライブラリ等で配信開始日を確認する。同カテゴリで3ヶ月以上回り続けている広告＝広告経済性が成立している最強の証拠。逆に、どの競合も広告を継続できていないカテゴリは経済性の危険信号
2. **レビュー増加速度**: 競合のレビュー数÷販売期間で月間増加数を出し、レビュー投稿率1〜2%と仮定して月間販売数を概算する（月間レビュー増 × 50〜100 ≒ 月販）
3. **ランキング持続性**: Amazon/楽天のカテゴリランキングに競合が居座り続けているか（瞬間風速か、定番化か）
4. **クラファン実績**: 類似テーマの調達額・支援者数
- 上記のCPC/CVR目安を試算に使う場合は、TikTok Creative Center等の公開ベンチマークと「同カテゴリで実際に回っている広告の存在」で裏を取る。**裏の取れない定数だけの試算は「未検証」と明記する**

## 共通キル基準（idea-kill-test / market-research で使用）
以下のいずれかに該当し、**証拠付きの反証**ができない場合はNo-Goとする:
1. Amazon/楽天で¥3,000以下の類似品が検索上位にあり、その低評価レビューに攻め筋（明確な不満）がない
2. 検索ボリューム・広告ライブラリ・クラファンのいずれにも需要の証拠が見つからない
3. 悲観シナリオ（CPC高め × CVR 1%）でCPAが粗利を超える
4. 薬機法・景表法により、核となるベネフィットが広告で表現できない
5. 原価率30%で製造・仕入れが現実的に成立しない
