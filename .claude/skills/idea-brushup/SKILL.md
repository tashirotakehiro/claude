# アイデアブラッシュアップ担当（マーケ目線付き）

以下の商品アイデアを「売れる企画」にブラッシュアップしてください。

**商品アイデア:** $ARGUMENTS

## 事業モデル
- D2C EC。SNS広告→LP→購入
- 商品単価¥10,000〜¥20,000、原価率30%、広告費40%
- Amazon/楽天の安い類似品に流れない独自性が必要
- 年間3,000万円以上の売上目標

## やること

### 1. 差別化の核心
- Q1: Amazonで類似品を検索した時に出る商品と何が違う？
- Q2: その違いに¥10,000〜¥20,000の価値がある理由は？
- Q3: 一言で差別化を表現すると？

### 2. 30文字以内のベネフィット案を7つ
- タイプ別（直球/好奇心/Before After/数字/共感）で出す

### 3. 懸念点5つと、LP上での解消方法

### 4. プロジェクト概要（各項目2〜5行で記入）
以下の項目を埋めてください。Notionのプロジェクトページに記載する内容です。

- **商品名（仮）**: 商品名を提案
- **開発コンセプト（何を目的とした商品なのか）**: 一言+補足2〜3行
- **テーマの本質（なぜこの商品が必要なのか）**: アイデア担当のSTEP 0分析を引き継ぎ、顧客の根本的な欲求・心理・既存市場の構造的問題点・尖らせるべきポイントを2〜5行で記載
- **主な機能・性能（どのように目的を達成するのか）**: 具体的な仕組み・構造
- **顧客のメリット（購入した人は何ができるようになるのか）**: 「〇〇によって△△できる」形式で
- **想定ターゲット（誰の悩みを解決するのか）**: 「〇〇の悩みを持った人」形式で
- **競合商品（悩みを抱えた人が他に検討する商品）**: 3商品。各商品のURL・価格・機能・差別化ポイント
- **学術的に言える効果（裏付け・信憑性）**: 商品が謳う効果に学術的根拠（メタアナリシス・RCT・レビュー論文等）があるかを ◎/○/△/× で評価し、根拠URLを添える。「誠実に言える効果の範囲」と「薬機法・景表法上、言ってはいけない表現」を明記する（例:「痩身・部分痩せは×。脚のだるさ・巡りケアは○。"痩せる/血行促進"は雑貨ではNG」）。健康・身体効果を謳う場合はWeb検索でエビデンスを確認すること。※学術評価は最重要ではなく1つの判断材料だが、「言えない効果を広告で言う」のは法的リスクなので必ず線引きする。
- **仕入価格（推定）**: 原価率30%から逆算した目安
- **販売価格（推定）**: 推奨価格と根拠

### 5. マーケ判定
- ターゲット市場の規模感（大/中/小）
- 競合の強さ（強い/普通/弱い）
- 広告映え度（高/中/低）
- 学術的な裏付け（◎/○/△/×）と、誠実に言える訴求軸（※最重要ではないが、広告表現の法的リスクに直結するため必ず併記）
- 総合: 「マーケ調査に進めるべきか」のYes/No推奨と理由
