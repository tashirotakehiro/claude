X投稿をClaudeで生成する際、`persona.md` と一緒にコンテキストとして渡す。

## 1. アカウント設計

- **アカウント名**: タカシ｜家庭内稟議研究所
- **プロフィール文**（160字以内・note導線を明記）:
  > 会社の稟議は通せるのに、家庭の稟議が通らなかった男。朝帰り→3週間の冷戦から10年、月の飲み会を1回→4回にしつつ夫婦仲を改善した「家庭内稟議」を研究中。42歳、法人営業、二児の父。ノウハウはnoteに全部書いてます↓
- **固定ポスト**: 原点の失敗談スレッド（下記④型）＋リプ欄にnoteリンク。noteが未発売の期間は失敗談スレッド単体
- ヘッダー画像: 「家庭内稟議書」を模したネタ画像（承認印の欄に「妻」）
- 顔出しなし。アイコンはイラスト

## 2. ポストの型（5種）と配合

| 型 | 目的 | 頻度 |
|---|---|---|
| ① あるある共感型 | 拡散・インプ稼ぎ | 毎日1本 |
| ② 稟議Tips型 | 有益性・保存 | 週3本 |
| ③ 妻側視点リスペクト型 | 炎上予防・女性層の味方化 | **週1本以上（必須）** |
| ④ 失敗談スレッド型 | 信頼・プロフ誘導 | 週1本 |
| ⑤ 実績・社会的証明型 | 販売（ローンチ後のみ） | 週2本まで |
| ⑥ 相談回答型 | ネタの無限化・エンゲージ | 週2本（相談が来始めたら主力化） |

### 型別の例文（生成時のお手本としてClaudeに渡す）

**① あるある共感型** — ディテールで刺す。オチは自虐か発見
> 飲み会の日の夫の帰宅は、玄関のドアを閉める音でわかる。音を立てないよう静かに閉めるほど、本人に自覚がある。俺は昔、閉めた記憶すらなかった。

**② 稟議Tips型** — ビジネス用語×家庭の翻訳が芸
> 飲み会の申請は「行っていい?」で出すな。それは白紙の稟議書だ。日付・帰宅時刻・予算・誰と・なぜ俺が行く必要があるか。会社で通る書式は、家庭でも通る。

**③ 妻側視点リスペクト型** — 「うちの夫に読ませたい」と思わせる
> 妻の「別にいいけど」を許可だと思ってる人へ。あれは「あなたが忘れてる論点が3つあるけど、自分で気づく?」という差し戻し通知。決裁者は理由を全部説明してくれない。会社と同じ。

**④ 失敗談スレッド型** — 原点の話を数字固定で（4,200円・未読・3週間）
> 結婚3年目、俺は妻に3週間口をきいてもらえなかった。原因は送別会。「1軒だけ」と言って出て、帰宅は午前2時。財布を落とし、玄関で妻にタクシー代4,200円を借りた。今日は、そこから月4回飲みに行けるようになるまでの話をする。🧵

**⑤ 実績・社会的証明型** — 数字と感想の引用のみ。煽らない。**実在の数字・実在の感想以外は絶対に書かない**（strategy.md 9. 誠実性ガードレール）
> 「家庭内稟議」note、発売3日で50部に到達しました。明日から980円に値上げします。一番嬉しかった感想は「妻に見せたら爆笑して、そのまま読まれた」でした。

**⑥ 相談回答型** — 相談要約→稟議用語で診断→処方箋。正典の症例（F-◯/S-◯）に接続すると世界観が締まる
> 【相談】「ソロキャンプに行きたいが言い出せない」。診断: これは否決を恐れて申請自体を出せていない**未提出案件**。稟議は出さなければ通過率0%です。まず月初の一括申請に1行で混ぜてください。単独で出すから重くなる。
> ※相談は特定を避けるため再構成しています。相談窓口はリプ欄へ。

## 3. 投稿スケジュール

- 1日2〜3本。**平日昼12時台**＋**夜21〜23時**
- このジャンルのゴールデンタイム: **金曜夜**（飲み会帰り・行けなかった夜の共感）と**日曜夜**（翌週の飲み会を通したい層）。①④型はここに置く
- 週次バッチ: Claudeで20本生成 → 7〜10本選別 → 実体験ディテールを最低1箇所注入 → 予約投稿

## 4. 導線ルール

- **ポスト本文に外部リンクを貼らない**（インプレッションが下がる通説のため）。noteリンクは**リプ欄にぶら下げる**
- 伸びたポスト（目安: 平均の5倍インプ）には24時間以内にリプでnoteリンク追加
- プロフィール→固定ポスト→noteの動線を常に整備
- **匿名相談窓口（マシュマロ等）を開設**し、プロフィールと⑥型ポストのリプ欄に常設。相談募集ポストを週1で定期投稿（「あなたの通らない稟議、診断します」）。相談はネタの供給源なので導線はnoteと同格に扱う

## 6. 炎上予防チェックリスト（全投稿に適用）

- [ ] 「嫁」ではなく「妻」
- [ ] 妻・女性を攻略対象/敵として書いていない（倒す敵は「気まずさ」「無計画」）
- [ ] 「女は」「妻というものは」等の一般化がない
- [ ] 嘘・ごまかしの推奨がない
- [ ] **妻本人がこのポストを読んだら?** → 笑うならOK、傷つくなら削除
- [ ] 女性がこのポストを引用RTするとしたら何と書くか想像した → 「わかる」「夫に読ませたい」なら合格
- [ ] 育児・家事の負担を軽視する表現がない（飲みに行ける前提には妻の負担があることを常に認める）

### 燃えかけた時の対応
- 反論しない。正しい指摘なら即削除＋訂正。曲解なら放置（引用RTで戦わない)
- ③型（妻側視点）の在庫ポストを平常運転で流す

## 7. リプライ・交流ルール

- 同ジャンル（パパ垢・サラリーマン垢）の伸びてるポストに1日2〜3件、芸風どおりのリプ（宣伝はしない。認知だけ取る）
- 自分のポストへのリプには初期は全返信（アルゴリズム上も有利）
- DM営業はしない。**相談もDMでは受けず、マシュマロへ誘導する**（匿名窓口なら距離を保てて、回答をコンテンツとして公開できる。DMで受けると再構成の断りを入れても燃えやすい）

## 8. 計測と改善（週次30分）

2. 勝ちポストを翌週のClaude生成プロンプトに「良い例」として貼る
3. プロフィールクリック率1%未満が2週続いたら、固定ポストを差し替え
4. 数字は `strategy.md` のKPI記録欄に追記
