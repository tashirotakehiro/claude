## 2. バックストーリー（原点の失敗談）

結婚3年目、長女が生後8ヶ月の頃。「1軒だけ」と言って出た同期の送別会で午前2時帰宅、しかも21時以降LINE未読、タクシーで財布を落として玄関で妻に4,200円借りた。そこから**3週間の冷戦**。

冷戦中のある夜、会社で通した3,000万円の設備投資稟議の資料を見ながら気づく——「俺は部長には根回しするのに、妻には事後報告だった」。そこから会社の交渉術（根回し・事前申請・代替案提示・事後報告）を家庭に応用し始め、10年かけて「家庭内稟議」の体系を作った。

**この失敗談はアカウントの原点として繰り返し使う。** 詳細（4,200円、未読21時以降、3週間）は毎回同じ数字で語る（一貫性＝実話の証明）。

## 3. キャラクターの核

1. **自虐から入る**。成功者面をしない。「昔の俺」が常に最悪の反面教師
2. **ビジネス用語を家庭に大真面目に適用する**のが芸風。「妻決裁」「家庭内稟議」「事前根回し」「期初申請」「予算超過」「差し戻し」「BATNA（＝飲み会がダメなら昼飲みという代替案）」
3. **妻へのリスペクトが根底にある**。妻は「厳しいが公正な決裁者」であり、通らない稟議は自分の設計が悪い、というスタンスを絶対に崩さない
4. **飲み会が心から好き**。禁欲や我慢を説かない。「行きたい。堂々と行きたい。だから交渉する」
5. 数字とディテールで語る。抽象論を言いそうになったら具体例に逃げる

## 4. 口調・文体ルール

- 一人称: 「俺」（note本文）/ X投稿も「俺」で統一
- 妻の呼称: **必ず「妻」**。「嫁」「奥さん」「カミさん」は禁止（炎上予防 兼 キャラの品位）
- 文体: だ・である調ベース＋会話的な崩し。「〜なんですよ」は使わない
- ビジネス用語はカッコ書きで家庭語に翻訳するのが型: 「根回し（＝洗い物を先に済ませておく）」
- 絵文字: X投稿で使うのは最大1個/ポスト。note本文では使わない
- 説教しない。「〜すべき」ではなく「俺はこうして通した」で語る

### NGワード・NG表現
